# Telegram-бот с регистрацией пользователей

Асинхронный Telegram-бот для регистрации пользователей с сохранением данных в базу данных SQLite.

## Возможности

- Регистрация пользователей через команду `/reg`
- Просмотр информации о пользователе через команду `/me`
- Валидация введённых данных (имя, возраст, город)
- Асинхронная работа с использованием `aiogram` и `aiosqlite`
- Хранение данных в базе данных SQLite

## Требования

- Python 3.7+
- Библиотеки: `aiogram`, `aiosqlite`, `python-dotenv`

## Установка

1. Клонируйте репозиторий:
   ```bash
   git clone <url-репозитория>
   cd <папка-проекта>
   ```

2. Установите зависимости:
   ```bash
   pip install -r requirements.txt
   ```

3. Создайте файл `.env` и добавьте токен вашего бота:
   ```env
   BOT_TOKEN=ваш_токен_бота
   ```

4. Запустите бота:
   ```bash
   python bot_stage3.py
   ```

## Команды

- `/start` — приветственное сообщение
- `/reg` — начать процесс регистрации
- `/me` — показать информацию о зарегистрированном пользователе

## Структура проекта

- `bot_stage3.py` — основной файл бота
- `requirements.txt` — список зависимостей
- `.env` — файл конфигурации с токеном бота
- `users.db` — база данных пользователей (создаётся автоматически)
